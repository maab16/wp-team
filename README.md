#Use Shortcode

	* Intialize - [wp-team]
	* Dots Navigation - [wp-team dots="true"]
	* autoplay - [wp-team autoplay="false"]
	* infinite slides - [wp-team infinite="true"]

##Use multiple attributes

```
[wp-team title="Your title" autoplay="true" dots="false"]
```
